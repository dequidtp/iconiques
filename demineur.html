Comment jouer ?

Le plateau cache des mines 💣. Chaque case révélée affiche le nombre de mines dans ses 8 voisines. À toi de déduire où elles se cachent sans jamais en creuser une.

  • Tap / clic en mode ⛏ Creuser : révèle une case (le premier clic est toujours sûr).
  • Bascule sur 🚩 Drapeau pour marquer une case suspecte. Sur ordinateur : clic droit pose un drapeau directement.
  • Tap sur un chiffre déjà révélé : si tu as posé autant de drapeaux que le chiffre, ça révèle automatiquement les autres voisines (le « chording », un gros gain de temps).
  • Le compteur 🚩 indique les mines restantes (mines totales − drapeaux posés).

Tu gagnes quand toutes les cases sans mine sont révélées.
